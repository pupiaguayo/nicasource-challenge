## Run Locally

Clone the project

```bash
  git clone https://github.com/pupiaguayo/nicasource-challenge
```

Go to the project directory

```bash
  cd nicasource-challenge
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
